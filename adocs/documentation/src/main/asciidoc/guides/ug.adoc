== Users' Guide

This users' guide introduces the xref:ug.adoc#_ug_core-concepts[core concepts] and ideas behind Apache Isis,
and tells you how to xref:ug.adoc#_ug_getting-started[get started] with a Maven archetype.

It also provides xref:ug.adoc#_ug_how-tos[how-to]s and a catalog of xref:ug.adoc#_ug_faqs.adoc[FAQ]s.


Other user guides are:

* the xref:ugvw.adoc[User Guide for Wicket viewer]
* the xref:ugvro.adoc[User Guide for Restful Objects viewer]


The guide also explains how to configure xref:ug.adoc#_ug_security[security], principally involving Apache Isis' integration with Apache Shiro.

xref:ug.adoc#_ug_more-advanced[more advanced] guidance on writing maintainable larger applications.


Later chapters discuss essential topics such as xref:ug.adoc#_ug_testing[testing] and how to xref:ug.adoc#_ug_deployment[deploy] your app, and discuss other ways in which you can xref:ug.adoc#_ug_extending[extend] or adapt the framework itself to your particular needs.

This users' guide is _not_ intended as a reference manual; for that see the *xref:rg.adoc#_rg[Reference Guide]*.  This guide also doesn't explain how to setup your development environment; for that see the *xref:cg.adoc#_cg[Contributors' Guide]*.
