== Fundamentals

This guide introduces the xref:ug.adoc#_ug_core-concepts[core concepts] and ideas behind Apache Isis,
and tells you how to xref:ug.adoc#_ug_getting-started[get started] with a Maven archetype.

It also describes a number of xref:ug.adoc#_ug_how-tos[how-to]s, describes how to influence the
xref:ug.adoc#_ug_object-layout[UI layout] of your domain objects (this is ultimately just a type of metadata), and it
 catalogues various xref:ug.adoc#_ug_faqs.adoc[FAQ]s.

Other user guides are:

* xref:ugvw.adoc#[Wicket viewer]
* xref:ugvro.adoc#[Restful Objects viewer]
* xref:ugsec.adoc#[Security]
* xref:ugtst.adoc#[Testing]
* xref:ugbtb.adoc#[Beyond the Basics]

Other reference guides are:

* xref:rgant.adoc#[Annotations]
* xref:rgsvc.adoc#[Domain Services]
* xref:rgcfg.adoc#[Configuration Properties]
* xref:rg.adoc#[Classes, Methods and Schema]

This users' guide is _not_ intended as a reference manual; for that see the *xref:rg.adoc#_rg[Reference Guide]*.  This
guide also doesn't explain how to setup your development environment; for that see the *xref:cg.adoc#_cg[Contributors' Guide]*.
