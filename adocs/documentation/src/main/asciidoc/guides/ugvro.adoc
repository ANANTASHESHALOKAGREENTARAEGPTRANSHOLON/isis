== User Guide for Restful Objects Viewer

Apache Isis' Restful Objects viewer is an implementation of the link:http://restfulobjects.org[Restful Objects spec],
which defines a generic way to expose a domain model through a REST (or more precisely, hypermedia) API.

This user guide discuss features, configuration and how to extend the Restful Objects viewer.

Other user guides are:

* the xref:ugb.adoc[User Guide Basics]
* the xref:ugvw.adoc[User Guide for Wicket viewer]
