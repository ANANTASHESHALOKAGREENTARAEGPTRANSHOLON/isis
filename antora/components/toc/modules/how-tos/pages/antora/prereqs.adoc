= Prereqs
include::_attributes.adoc[]

These are the prereqs for using Antora to generating the website:

* Node LTS (10.x)
+
[source,cmd]
----
choco install node-lts
----

* use `npm` to install Antora itself:
+
[source,bash]
----
npm install -g antora
----

* use `npm` to install Serve (a node-based mini webserver):
+
[source,bash]
----
npm install -g serve
----
